bunsanweb is a decentralized web born in Tokyo which forms an open network of individual programs that freely share information between them.
Its foundation is built on standard browser-side web technologies.

Mission of bunsanweb

We call it “bunsanweb”

Features

  • Endpoint-scripting:
    • mixes both client-side and server-side scripting features to access and produce rich Web resources
  • Universal event stream:
    • for allowing unspecified endpoint scripts to communicate without depending on specific centric channels
  • Endpoint-relative hyperlinked space:
    • each endpoint views resources from each local linked resouce to the universal on Web; e.g. your "personal data" is also a relative resource and it may link to "friends" on the universal web.

History

bunsanweb was started a from a desire to create an environment in which programs could use each other as resources in a loosely-coupled manner like on the worldwide web.
